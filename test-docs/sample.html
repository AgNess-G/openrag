Pipeline Stages

The pipeline processes documents through five stages: parse, preprocess, chunk, embed, and index.

Stage Protocol Implementations
Parse DocumentParser Docling, PlainText, MarkItDown, Auto
Chunk Chunker Recursive, Character, Semantic, PageTable, DoclingHybrid
Embed Embedder OpenAI, Ollama, WatsonX, HuggingFace

Getting Started

Configure the pipeline via config/pipeline.yaml or use a preset from config/presets/.
